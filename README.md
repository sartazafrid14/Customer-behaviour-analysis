# 📊 Customer Behavior Analysis

A data analytics project built using **Python** and **Jupyter Notebook** to analyze customer purchasing behavior, identify trends, segment customers, and generate business insights through data visualization and machine learning techniques.

---

## 📌 Project Overview

This project analyzes an e-commerce customer dataset to understand customer demographics, purchasing patterns, payment preferences, and churn behavior. The dataset was cleaned, explored, and visualized to provide actionable insights that can help improve customer retention and business performance.

---

## 🎯 Objectives

- Clean and preprocess customer data
- Perform Exploratory Data Analysis (EDA)
- Analyze customer purchase behavior
- Segment customers using RFM Analysis and K-Means Clustering
- Identify customer churn patterns
- Generate business insights and recommendations

---

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
